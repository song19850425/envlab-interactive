1.7板块一 · 政策性资金申报

国家发改委节能降碳专项

★★☆☆☆ 50-80万
场景 SCENE

场景

碳不是突然压在头上的石头，而是十年前就开始滚下来的雪球。但很多人等雪球砸到眼前，才发现自己没穿鞋。

某西部能源城市，GDP 1800亿，其中煤化工贡献了将近一半。市委书记2025年上任时最头疼的数字不是GDP增速，而是单位GDP能耗——全省倒数第一，是全国平均水平的2.3倍。

谁在焦虑？

- 市委书记：2025年省政府工作会上，省长当着一百多号人的面念了各市能耗排名——念到本市名字时，会议室安静得能听见空调声。会后分管副省长拍了拍书记的肩膀："抓紧降，不然明年新上的煤制油项目，怕是不好批。"

- 市发改委主任：知道有"节能降碳专项资金"，但申报条件是"绿色低碳先进技术示范工程"。本市最大的煤化工企业能耗确实大，但用的是十年前的技术，跟"先进"完全不沾边。硬改技术路线？一条生产线3个亿，企业疯了吗？

- 煤化工企业总工程师：能耗指标被卡得死死的，扩产不能放、新项目不能批。不是不想降碳，是不知道怎么降——CCUS（碳捕集）技术一条线投资1.5亿，捕一吨碳成本300块，没有收益模型支撑，董事会根本不会批。

政策底色：国家发改委节能降碳专项每年安排中央预算内投资约50亿元。"绿色低碳先进技术示范工程"可获最高30%的投资补助，是几项专项中补助比例最高的。

痛点 PAIN

痛点

节能降碳不是"少用电"这么简单。它有技术门槛、有财务门槛、有政策门槛——三者勾连形成一个"动了怕死，不动等死"的僵局。
利益相关方核心痛点深层原因
地方政府能耗"双控"压力巨大产业结构偏重，转型代价高、阵痛期长
高耗能企业能耗指标限制产能扩张节能降碳技术投资大，但收益难以量化和内部化
发改委好项目太少大多数项目"降碳不先进"或"先进不降碳"
技术服务商技术好但卖不出去企业与技术之间的"翻译"缺位
咨询公司机会—**节能降碳技术筛选+经济性模型+30%补助申报**

致命细节：发改委评审有一个概念叫"示范"——你的项目必须能在全国可推广、可复制。很多企业把材料写成了"我有多困难"，而评审专家要看的是"我有多先进"。困难永远不是加分项，示范意义才是。我们帮客户做的第一件事，就是把"我要降碳"这个故事，重新讲成"中国XX行业降碳的样板"。

方案 SOLUTION

方案

节能降碳专项资金申报的核心，不是算降了多少碳——是证明你的降碳方法"别人也能用"。

我们五步申报法

```

① 碳诊断 → 企业碳排放全面盘查，识别最大减排空间

② 技术匹配 → 从13种推荐绿色低碳技术中筛选最优路径

③ 经济模型 → 建立"投资-节能-碳收益"三联动财务模型

④ 示范叙事 → 把项目包装为"XX行业绿色低碳技术示范工程"

⑤ 申报全流程 → 对标评分表逐项打磨+省级推荐+答辩演练

```

关键技术方法

- 企业碳盘查工具：基于ISO 14064的全流程碳排放核算

- 节能降碳技术适用性矩阵：13种技术 x 10个行业 = 130个应用场景匹配

- 碳资产模型：CCER开发可行性评估 + 碳交易收益预测

关键数据锚点

- 主管部门：国家发改委（环资司）

- 资金类型：中央预算内投资，单个项目 ≤ 1亿元

- 核心门槛：绿色低碳先进技术示范工程补助比例 ≤ 30%

- 窗口：每年9-10月（唯一窗口）

- 2026年：碳达峰行动方案全面落地，优先支持"低碳零碳负碳"技术示范

- 隐藏加分项：有CCER开发计划的项目，优先级明显更高

🔑 **销售话术锚点**："你们这条生产线一年排放110万吨CO₂，能耗指标把你们的新项目卡死了。我帮你们选一套绿色低碳技术路线，总投资假设3亿，发改委能帮你掏30%——9000万，无偿的。更重要的是，改造后的碳减排量可以开发CCER，一年大概再给你增加1500万的碳交易收入。三年回本，之后全是利润。"

目标客户画像

- 年碳排放≥50万吨的煤化工、钢铁、水泥、电解铝企业

- 有明确产能扩张需求但受困于能耗指标的企业

- 所在城市能耗"双控"排名靠后、有转型政治压力的地区

- 有意愿探索CCUS、氢冶金、短流程电炉等前沿低碳技术的龙头企业
