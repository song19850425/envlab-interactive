产业平台公司融资

场景 SCENE

场景

据不完全统计，截至2025年底，全国AA级以上城投/产投平台超过**1200家**，但其中真正完成了市场化产业转型、具备独立造血能力的，不到**20%**。剩下80%的平台，一边是被迫剥离政府信用后的融资困局，一边是被推着往产业方向转型的政治压力——两头受挤，苦苦挣扎。

故事从一个评级调到期的前夜开始：

华中某地级市产投集团，AA评级。五年前还是市政府眼里的"钱袋子"——城投债、银行贷款、信托通道，想融钱总有办法。但这两年情况大变：

- 隐债监管：2018年以来的隐性债务化解已经进入深水区，新增融资不得与政府信用挂钩；

- 评级压力：下个月评级机构进场年审，如果这次守不住AA，发债利率至少跳升200个基点；

- 化债任务：集团存量隐性债务还有12亿待化解，市里下的死命令是两年内清零；

- 产业使命：市里要求集团转型为产业投资平台，但产业投资项目需要钱——这就陷入了一个死循环：不化债融不到资，不融资做不了产业，不做产业化不了债。

财务总监在办公室熬了三个通宵，做出来的方案被董事长否了两次。核心问题就一个：能不能找到一种钱，既帮我们化解存量债务（30%），又支持新产业投资（70%），而且这笔钱的成本不能太高——因为产业投资回报周期长，资金成本高了就是"借新还旧"，不可持续。

他听说有一种定向资金可以做到：单笔5-8亿，其中30%用于化债（置换高息负债），70%用于产业投资——核心逻辑是"以产业收益反哺化债成本"。但他不知道从哪找这种资金、怎么设计交易结构、以及如何说服出资方。

AA平台的中国式困局：手上有资产、政府有使命、市场有机会，但就是缺一笔"设计过的钱"。
痛点 PAIN

痛点

平台公司不缺资产，缺的是"搭配好用途的资金"。
利益相关方核心痛点深层原因
产投集团化债和投资两头吃紧资金用途被割裂——化债的钱不让投资，投资的钱不帮化债
地方政府平台的债就是政府的"影子债"不能明着兜底但必须暗着操心
出资方既要收益又要安全纯化债怕打水漂，纯投资怕回报太慢
评级机构平台资产质量持续恶化经营性资产少、隐形债务多
咨询公司机会—**"30%化债+70%产业"的资金结构设计和交易撮合**

致命细节：大部分平台的思维还停留在"先融钱再想怎么用"的阶段。但在当前的监管环境下，出资方要求的是"以终为始"——你必须先讲清楚产业投资的退出路径和回报预期，让对方相信你的70%能产生足够收益来覆盖那30%化债的风险溢价。

深层原因：这是一个"三方信任困局"——政府想让平台转型但不给钱、平台想融钱但信用被剥离、出资方有钱但信不过平台的产业能力。谁能同时获得三方的信任，谁就能做这笔交易的设计者和撮合者。

咨询机会：核心不是帮客户"找到钱"，而是帮客户"设计一个出资方愿意接受、政府愿意背书、平台能执行的交易结构"。这个结构里，化债和投资不是两条平行线，而是一个完整的闭环。

方案 SOLUTION

方案

不是找钱，是设计钱的流向。

五步工作流

```

① 平台画像 → 全面审计平台资产负债结构

识别可用于产业投资的经营性资产

量化化债缺口和产业投资缺口

产出：《平台资产负债诊断报告》

产出：《产业投资可行性研究》+ 投资逻辑报告

③ 资金结构设计 → 设计"30%化债+70%产业"的资金分配方案

化债部分：置换高成本融资（信托/融资租赁）

产业部分：股权投资+产业基金LP

产出：《资金使用方案》+ 交易结构设计

④ 还款路径建模 → 构建"产业收益→覆盖化债利息→返还本金"的现金流闭环

关键：保守测算产业投资回报，做压力测试

产出：《还款能力测算报告》

⑤ 出资方对接 → 根据项目特征匹配定向资金渠道

协助路演、尽调、谈判

产出：投资协议 + 资金到账

```

关键技术手段

- 化债工具箱：合规的债务置换路径设计（避免触发新增隐性债务红线）

- 产业投资筛选模型：基于区域比较优势和产业梯度的投资方向论证

- 现金流闭环设计：确保产业收益能够真实覆盖化债成本的结构设计
