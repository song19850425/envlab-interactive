商务对接与协调

★★☆☆☆ 按日收费 | **客户粘性**：中 | **备注**：关系导向
场景 SCENE

场景

一个县级项目要走的审批环节：县局→县发改→县政府常务会→市局→市发改→省厅处室→省厅分管领导→国家部委（部分项目）。每一个环节都可能"卡住"，而每卡一次平均延误**2-4个月**。很多项目不是死在评审上，是死在"迷路"上。

谁在焦虑？

- 县发改局局长：攒了大半年的项目终于过常务会了，但要报到省厅却不知道找谁。打了一圈电话，每个处室都说"你先放着"——究竟哪个处室是主办？哪个是协办？谁有话语权？

- 项目执行单位负责人：一边是项目推进的KPI，一边是跑部门的无穷会议。一个月去了省城八趟，请客吃饭花了好几万，事情只推进了"一小步"。

- 县委书记：时间就是资金。中央转移支付的盘子每年都有变化，窗口期一过资金就归零。项目推不动不是没钱——是缺能把各个环节打通的人。

痛点 PAIN

痛点

在中国做项目，最难的不是技术和资金，是"人"和"关系"。
利益相关方核心痛点深层原因
项目申报单位不知道找谁、不知道怎么找政府内部职能交叉，外部人很难理清权责边界
县领导过程不可控、时间不可预期靠基层人员自己跑，效率极低且结果不确定
省级审批部门收到的材料质量参差不齐基层单位缺乏与上级部门预沟通的渠道和能力
咨询公司机会—**关系网络 + 专业能力 = 加速器**

致命细节：很多项目在"等消息"中错过了最佳窗口期。不是处理中，是真的被压在某个处室的文件堆里没人看——处长太忙了，排不上优先级。有关系的单位打个电话就有人从文件堆里翻出来，没关系的就只能等。

方案 SOLUTION

方案

我们不是"找关系"，是"建通道"。把专业能力和关系网络结合起来，让项目在正确的轨道上以正确的速度推进。

五步对接推进法

```

① 审批路径测绘 → 画出项目从立项到获批的完整审批节点图，标注每个节点的主办/协办单位

② 关键人对接 → 基于已有的省级/部级关系网络，为每个关键节点匹配对接人

③ 材料预沟通 → 在正式报送前，将核心材料提交关键处室做"非正式预审"（这是最有价值的环节）

④ 问题提前消化 → 预沟通中收集的修改意见，在正式报送前全部消化掉

⑤ 节奏跟催 → 建立审批进度台账，按节点主动跟进，确保材料不"沉底"

```

关键方法论

- 审批路径可视化：用甘特图和流程图呈现每个项目的审批全景，让客户看到"我们走到哪了""下一步是什么"

- 预沟通机制：正式材料报送前必须先做预沟通——这不仅是效率策略，更是免责策略（预沟通过的意见已在修改中采纳）

- "三点一线"协调法：县-市-省三级联动，确保下级的诉求能准确传递到上级，上级的意图能被下级正确理解

- 日报周报制度：对接进度每日简报、每周详报，让决策层掌握节奏

关键数据锚点

- 收费标准：按日计费，日费率 5000-15000元/天（根据对接难度和专家级别）

- 典型周期：单个项目对接周期 1-3个月（根据审批路径长度）

- 核心价值：项目推进速度提升 50%-100%，关键节点延误风险降低 70%

- 服务模式：可整包（按月）也可零包（按天），灵活匹配客户需求

- 特殊价值：预沟通环节是其他任何服务商无法替代的——这完全基于历史积累的关系和信任

价值 VALUE

价值

对客户的三重价值

价值维度具体收益量化预期
速度项目推进不卡壳审批周期缩短30-50%
确定性从"等消息"到"知道走到哪了"100%过程透明，关键节点有预告
风险管控提前消化修改意见正式报送一次性通过率大幅提升

对我们的商业价值

- 收入模式：按日收费，单个项目 15-50万元（1-3个月周期）

- 利润率：高（核心成本是专家时间，无硬件投入）

- 客户粘性机制：信任一旦建立，后续项目默认找同一团队做对接——换人等于重新建关系

- 延伸机会：对接过程中发现的融资需求→平台公司融资（业务4.7）、对接过程中发现的化债需求→政府化债（业务4.6）

- 竞争壁垒：关系网络需要时间积累，是最难以复制的竞争壁垒

行动 ACTION

行动

客户开发路线

时间节点行动项责任人
**日常**维护省级/中央部委关键处室的关系图谱，定期更新组织架构和人事变动关系团队
**项目启动前**与客户明确对接范围、关键节点和预期周期，签订服务协议商务团队
**启动后第1周**完成审批路径测绘和关键人匹配技术团队
**启动后第2-3周**完成首轮预沟通，收集反馈意见对接专家
**后续持续**跟催+简报，直至项目获批项目负责人
💡 **销售话术锚点**：
- "上次你们报省厅的项目，卡了多久？知道卡在谁那儿了吗？"（引发痛点回忆）
- "你们自己去跑，一个月跑八趟还在原地踏步。我们有固定的沟通渠道，一个电话就能把材料从文件堆里翻出来。"
- "预沟通不是走后门，是让审批部门提前了解你的项目——等到正式评审时，你的材料已经是"改了N版的优化版"，而对手的还是"第一次见到的原始版"。"

理想客户画像

- 目标：有重大立项审批需求但缺乏省部级对接渠道的县级政府/企业

- 特征：曾经自己跑过审批流程但屡次碰壁，深刻体会过"没人引路"的痛苦

- 触发点：新项目立项审批启动、旧项目在某个环节卡住超过1个月

- 决策人：项目主管单位一把手或县领导
