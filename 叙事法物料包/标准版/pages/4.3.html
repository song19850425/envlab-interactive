答辩与评审辅导

★★★☆☆ 成功费为主 | **客户粘性**：高 | **备注**：高价值
场景 SCENE

场景

某省厅答辩室，每位申报单位的陈述时间是**15分钟**，专家提问10分钟。25分钟定生死。台上的县长手心全是汗，PPT翻到第三页就卡住了——专家突然问了一句"你们的收益测算基准年为什么选2019年而不是2020年？"县长愣住了，旁边局长小声说"我们……没想过这个问题。"

谁在焦虑？

- 即将答辩的县长：材料不是他写的，数据不是他算的，但台上那15分钟必须他来讲。背熟稿子已经用了三个通宵，但专家提问完全不按稿子走。

- 发改局局长：准备了三个月，材料改了七版，好不容易过了初筛进了答辩。如果答辩翻车，前面所有努力白费。更糟的是——被淘汰的项目名单会公开，面子挂不住。

- 省级部门对接处室：每年要推荐项目上去，但答辩表现直接影响省里的名额分配。省里丢不起这个人——一个答辩翻车可能影响全省下一年的名额。

痛点 PAIN

痛点

不是讲不好，是没准备好。没有经历过真实答辩的人，永远无法想象专家会从什么角度提问。
利益相关方核心痛点深层原因
答辩人（县长/局长等）临场紧张、应答失当缺乏答辩经验，不知道"该说的"和"该避开的"
项目申报团队不了解评审关注点做了几个月的材料但没和真正的评审专家对谈过
省级推荐单位担心推荐的答辩翻车答辩结果直接影响省级名额和声誉
咨询公司机会—**信息差 + 经验复制 = 不可替代的高价值服务**

致命细节：答辩中最致命的错误不是答错，是"说实话"。比如专家问"项目有什么风险"，县长老老实实说"征地拆迁可能有阻力"——这在评审语境里等于"项目不可行"。你应该说"已建立三项征地保障机制，确保按期交付"。

方案 SOLUTION

方案

我们不是在"排练"，是在"精准预测并提前准备所有可能的攻击角度"。

六步答辩辅导法

```

① 竞争对手情报收集 → 研判同批次竞争项目的优劣势，建立"攻击-防御"策略库

② 答辩材料重构 → 15分钟PPT按"痛点-方案-差异-效益-承诺"五段式重构

③ 专家问题库预判 → 模拟50+个可能提问，分为"常规题、刁难题、陷阱题"三类

④ 逐题打磨话术 → 每个问题准备3层应答：标准答案→补充论据→收尾金句

⑤ 三轮模拟答辩 → 一轮脱稿讲、二轮压力问（打断、质疑、追问）、三轮全真模拟

⑥ 现场陪跑 → 答辩当天场外待命，最后一轮复盘+心理建设

```

关键方法论

- 攻击面管理法：提前梳理项目所有弱点（数据不实、逻辑断裂、前置条件未满足等），每个弱点准备防御话术

- ABC应答法：A层（正面回应，一句到位）→ B层（补充数据/案例支撑）→ C层（回归主题，强化核心叙事）

- 情境预设法：模拟不同评审风格（学者型、政策型、实操型），训练答辩人切换语言体系

- 微表情与节奏控制：培训答辩人的语速、停顿、眼神和肢体语言

关键数据锚点

- 辅导周期：3-5天集中训练（根据答辩准备时间倒排）

- 收费模式：成功费制（答辩通过后按获批资金的一定比例）+ 基础辅导费 15-30万元

- 成功费比例：按获批资金额的 0.5%-2%

- 通过率：经完整辅导的项目答辩通过率 ≥ 85%

- 关键节点：答辩通知到正式答辩通常只有7-15天，必须提前签订辅导协议

价值 VALUE

价值

对客户的三重价值

价值维度具体收益量化预期
兜底保障把"临场翻车"的概率降到最低通过率从50%提至85%+
认知升级答辩人真正理解项目的核心叙事答完辩后能向任何人讲清楚这个项目
心理建设焦虑转化为信心三轮模拟后，自信面对任何提问

对我们的商业价值

- 单项目收入：20-100万元+（基础费+成功费），高杠杆模式

- 利润率：极高（核心成本是专家时间，边际成本低）

- 不可替代性：答辩辅导是最难被AI和标准化产品替代的环节——高度个性化、高度经验依赖

- 客户粘性：一次辅导建立深度信任，后续项目申报+答辩全链绑定

- 口碑效应：答辩通过后客户主动转介绍，同一省份多个项目快速铺开

行动
