3.2板块三 · 数据资产与平台转型

平台公司评级提升（2A→2A+→3A）

★★★☆☆ 80-150万+分成
场景 SCENE

场景

一个中部省份的地级市城投公司，会议室里气氛凝重。董事长刚接到通知：上一期中期票据的发行利率比同级别平台高了120个bp——就因为评级停留在2A，投资人用脚投票。

故事从一个"差一点"的困境开始：

某中部地级市城投，总资产312亿，2025年营业收入21亿。各项指标看起来"还可以"——但"还可以"在信用评级的世界里，意味着"不够好"。

评级公司给出的反馈很刺眼：

- 资产端：312亿总资产中，公益性资产占比47%，这些资产"在法律上不能用于偿债"；

- 收入端：21亿营收里，真正市场化收入不到6亿，经营性收入占比不足30%；

- 负债端：有息债务178亿，其中一年内到期62亿，短期偿债压力巨大；

- 治理端："政府平台"的标签太重，投资人担心政策转向后的偿债意愿。

2A的评级像一个透明的天花板——发债可以发，但利率上浮100-150bp，融资成本每年多出5000万-1亿。更要命的是，越来越多的金融机构内部规定"2A以下不投"，2A也只是勉强及格。

"我们不是没钱，是资产结构不对。"董事长的这句话，就是整个故事的起点。

转折的逻辑很简单： 如果能把47%的公益性资产压到30%以下，把市场化收入占比提升到50%以上，把短期债务占比优化到30%以内——2A到2A+，只需要这三件事。

痛点 PAIN

痛点

评级提升不是"做材料"，是一次资产结构和商业模式的深度手术。
利益相关方核心痛点深层原因
城投董事长/总经理融资成本居高不下，评级"差一口气"资产结构失衡是历史遗留问题，内部推不动
地方财政局/国资委既要保平台信用、又不能违规注资"43号文"后经营性资产注入路径受限
评级公司现有材料无法支撑评级上调平台提供的资料"数据多、逻辑少"，缺少资产管理专业叙事
金融机构/投资人想放款但风控过不了平台资产质量不透明，无法评估真实偿债能力

咨询公司的机会在哪？

评级提升不是一个"写报告"的活——它需要资产重组、产业导入、财务优化、治理升级四位一体的系统工程。平台公司自己不具备这种跨领域的整合能力。市场上能做"评级诊断"的很多，能做"资产重组"的也有，能把两者串起来、把评级真正推上去的——极少。这就是溢价空间。

方案 SOLUTION

方案

评级提升的本质，是把"政府融资平台"改写成"城市综合运营商"——让评级公司看到的是一个可持续经营的"企业"，而不是一个依赖财政输血的"壳"。

五步工作流

```

🔍 第一步：评级差距诊断

→ 对标3A/2A+平台的资产结构、收入结构、负债结构

→ 逐项测算各指标的"及格线"与"优秀线"

→ 产出：《评级提升差距分析报告》

🏗️ 第二步：资产重组方案

→ 剥离/置换低效公益性资产，注入经营性资产

→ 路径：经营权注入（不需要产权过户）、特许经营权、数据资产入表

→ 产出：《资产结构优化方案》

🏭 第三步：产业导入与收入重构

→ 围绕城市资源导入经营性业务：停车运营、充电桩、数据服务、水务一体化

→ 构建"市场化收入"的财务叙事

→ 产出：《产业导入与收入提升计划》

🏦 第四步：财务结构优化

→ 短债转长债、高息置换低息、隐性债务显性化

→ 搭建合理的"借新还旧"时间表

→ 产出：《融资结构优化方案》

📋 第五步：评级申报与路演

→ 编制评级申报材料，突出"城市综合运营商"定位

→ 组织评级公司尽调路演

→ 产出：《信用评级申报书》《投资价值分析报告》

```

价值维度具体收益量化预期
**融资成本骤降**评级每升一档，发债利率下降100-150bp以100亿存续债务计，年化节省5000万-1.5亿

**真实案例参照**：2024年，某东部省份城投通过注入停车、水务、数据三类经营性资产，优化债务结构，6个月内实现2A→2A+评级上调。评级提升后，当期20亿中期票据发行利率从5.8%降至3.9%，一次性节省发行成本3800万。2025年全国已有超过30家平台公司启动评级提升专项工作。
