建信产业基金对接

★★☆☆☆ 融资额1-2% | **客户粘性**：中 | **备注**：资源导向
场景 SCENE

场景

某地级市引进了一个新能源电池产业链项目，总投资**30亿元**。当地政府承诺了土地、税收优惠和配套基础设施，但最大的缺口在于：需要 **10亿元产业引导基金**做基石投资，才能撬动社会资本跟进。市里的产业引导基金盘子只有2亿，远远不够。而国家级建信产业基金的盘子是千亿级——但排队申请的地方有上百个，怎么挤进去？

谁在焦虑？

- 市招商局局长：项目签了意向协议，投资方要求3个月内产业基金到位，否则协议自动失效。市里几套班子急得团团转，但不知道建信基金的门朝哪开。

- 分管产业的副市长：这个项目是全市产业转型的关键一仗。隔壁市去年通过建信基金拿到了15亿，一举奠定了新能源产业龙头地位。自己市如果拿不到，项目可能迁到隔壁——那就不是丢一个项目的问题，是整个产业布局的崩盘。

- 项目建设方：他们的商业逻辑很清楚——拿到国家级基金的背书，银行授信就会大量跟进。但建信基金的投资决策流程、偏好和节奏完全是黑箱状态。

痛点 PAIN

痛点

国家级产业基金的对接，本质上是一场**信息极度不对称**的游戏。基金管理方只和"对的人"对话，而大多数地方连"对的人是谁"都不知道。
利益相关方核心痛点深层原因
地方政府有项目但对接不上国家级基金没有进入基金"投资视野"的渠道和背书
项目建设方等米下锅，时间压力巨大商业计划已确定，资金缺口是唯一瓶颈
建信基金管理方项目池过载、筛选成本高全国上千个项目排队，优先看"有信任背书的推荐"
咨询公司机会—**渠道资源 + 信任背书 + 符合基金口味的项目包装 = 关键缺口填补**

致命细节：建信基金不看可研报告，看的是"投资建议书"。两者逻辑完全不同——可研报告回答的是"这个项目是否必要和可行"，投资建议书回答的是"为什么要投这个项目而不是投别的"。绝大多数地方政府的申报材料写成了可研报告，在基金方眼中等于"文不对题"。

方案 SOLUTION

方案

建信基金对接不是"递材料"，是开一个"信息通道"——把地方的好项目翻译成基金听得懂的投资语言，通过信任渠道高效传递。

五步基金对接法

```

① 项目投资逻辑重构 → 将政府的产业故事转化为投资回报逻辑：行业空间→竞争优势→盈利模型→退出路径

② 基金偏好匹配 → 研究建信基金当期投资方向、区域布局、单笔偏好，做精准定位

③ 投资建议书编制 → 按PE/VC投资委员会评审标准编制（市场分析+竞争壁垒+财务预测+退出方案）

④ 渠道对接与路演 → 通过已有渠道将项目推送到基金管理方的核心决策层，组织专题路演

⑤ 尽调全程陪跑 → 协助应对尽调中的各种挑战性问题，确保项目不被中途拦截

```

关键方法论

- 投资故事翻译术：政府叙事→投资叙事，核心是将"产业价值"翻译为"资本回报"——基金不关心你创造了多少GDP，关心的是投入1元能赚回多少元

- 同频对话法：用基金方熟悉的语言体系沟通（IRR、DPI、退出机制、估值倍数），而非政府文件语言

- 渠道网络法：核心价值在于已建立的建信基金直接联系渠道——这是竞争对手短期内无法复制的

- 风险对冲包装：主动梳理项目风险点并提供对冲方案，在基金方眼中这是成熟项目方的标志

关键数据锚点

- 收费模式：按实际到账融资额的 1%-2% 收取成功费

- 典型案例：协助产业基金到位5亿元，收入500-1000万元

- 服务周期：3-9个月（从项目筛选到资金到位）

- 建信基金规模：管理资产超3000亿元，单笔投资数千万元至数十亿元

- 匹配方向：新能源、新材料、高端装备、数字经济等战略性新兴产业

- 前置条件：项目必须位于建信基金当期重点投资领域，且已有明确的项目主体和初步商业计划

价值 VALUE

价值

对客户的三重价值

价值维度具体收益量化预期
资金注入国家级资本直接进入基金投资数亿元至数十亿元
信用背书"建信投过的项目"提升融资能力银行跟进授信额度放大3-5倍
产业资源基金投后管理的产业资源对接供应链、客户、技术伙伴全面升维

对我们的商业价值

- 单项目收入：高天花板，典型项目收入 500-2000万元+

- 收费模式优势：纯成功费/阶梯成功费——客户前期没有成本风险，成交阻力低

- 竞争壁垒：建信基金的直接对接渠道是最高壁垒——不是谁都能拿到基金方的内部推荐通道

- 客户粘性：中等（单次融资成功后粘性不强），但一旦在一个地区建立成功案例，该地区的后续项目开发成本急剧下降

- 延伸机会：基金到位后的后续融资（业务4.7）、上市公司扶持政策（业务4.9）

行动 ACTION

行动

客户开发路线

时间节点行动项责任人
**日常**维护建信基金等国家级产业基金的渠道关系，跟踪当期投资方向和偏好资源团队
**客户接洽**通过地方政府招商部门、产业链龙头企业、行业协会等渠道发现标的项目商务团队
**签约前**对标的项目做快速投资逻辑诊断，判断是否匹配基金偏好——不匹配的坚决不签技术团队
**签约后**立即启动投资逻辑重构+投资建议书编制+渠道对接三项并行推进全团队
💡 **销售话术锚点**：
- "建信基金去年投了X省X市的项目，规模和你们差不多。他们能拿到的条件，你们完全可以复制——差别只在于"谁帮你推"。我们来推，就是帮你的项目跳过前置筛选，直接进决策层视野。"
- "你已经为这个项目花了几年时间，产业规划、土地、配套都到位了，就差最关键的10亿产业基金。这个10亿到了，项目就活了；到不了，前面所有投入都会贬值。"
- "融资市场上不缺钱，建信基金的账上趴着几千亿在找出路。缺的是能把好项目和好资金对接到一起的'翻译官'——这就是我们的角色。"

理想客户画像

- 目标：地级市或强县域政府招商部门、大型产业项目投资方

- 特征：项目总投资20亿元以上，属于国家战略新兴产业，有明确的落地诉求但资金缺口在5亿元以上

- 触发点：项目签约后资金筹集期；地方政府主要领导亲自督办的重点项目

- 决策人：分管产业的副市长、招商局局长、项目投资方董事长
