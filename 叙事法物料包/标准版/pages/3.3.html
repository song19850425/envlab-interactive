3.3板块三 · 数据资产与平台转型

产业型国企转型咨询

★★☆☆☆ 40-80万+分成
场景 SCENE

场景

2025年6月，国务院国资委一纸文件下发，要求各地城投类企业限期完成"335指标"核查。某西部省份城投公司董事长连夜开会——财务总监递上的报表显示，三项指标全部飘红。

故事从一道"死线"开始：

某西部地级市城投，总资产156亿。公司经营了十几年，身份一直是"政府融资平台"——这是它的生存方式，也是它的原罪。

国资委"335"指标的三把刀悬在头顶：

- 经营性资产占比≥30%：公司目前18%，差了12个百分点；

- 经营性收入占比≥30%：公司目前11%，主要靠财政补贴和代建管理费；

- 财政补贴占利润总额≤50%：公司目前是82%——也就是说，去掉财政补贴，公司亏损。

2026年底是核查大限。不达标怎么办？限批新项目、限发新债务、限期重组，甚至面临撤并。

"我们不是不想转型，是不知道怎么转。"董事长的困惑是普遍的：城投的基因是基建和土地，要变成"产业国企"，产业从哪来？

问题的本质是身份转换： 从"帮政府融资"到"帮政府经营城市"。停车场运营是产业，数据服务是产业，充电桩是产业，水务一体化是产业——问题不在有没有产业，在于能不能把这些散落的城市资源整合成一个可经营的商业实体。

痛点 PAIN

痛点

"城投转型"喊了十年，真正转成的没几个。不是缺意愿，是缺路径、缺能力、缺时间。
利益相关方核心痛点深层原因
城投管理层想转型但不知道方向在哪多年来以"完成任务"为导向，缺乏市场化经营基因
地方政府中央考核压力传导，但对平台"断奶"不放心平台仍是地方基建的主要抓手，不敢彻底放手
国资委/财政局既要考核达标、又要保持平台稳定"335"推得太急怕出事，推得太慢怕问责
城投员工担心转型后饭碗不保基建思维固化，缺乏产业运营能力

咨询公司的机会在哪？

"335转型"是硬约束、有时限、有明确的达标标准——这比一般的"战略咨询"好卖得多。客户不是"可做可不做"，而是"到时间必须达标"。而且转型方案一旦落地，后续的产业导入、资产注入、数据入表、融资服务都是复购机会。这是一个"入口型"业务——先通过335合规打开局面，再逐步导入高利润的深度服务。

方案 SOLUTION

方案

城投转型不是"另起炉灶"，而是把已经握在手里的城市资源重新定义、重新定价、重新经营。

六步转型工作流

```

📊 第一步：335差距诊断

→ 按国资委最新口径测算经营性资产、经营性收入、财政补贴依赖度

→ 逐项对标，识别"差在哪、差多少、怎么补"

→ 产出：《335指标达标差距诊断报告》

🗂️ 第二步：资产分类与重组

→ 将现有资产分为：经营性/准经营性/公益性三档

→ 准经营性资产（停车、水务、供热等）通过"定价+运营"提升为经营性

→ 公益性资产通过置换、剥离、经营权转化等方式出表

→ 产出：《资产分类重组方案》

💼 第三步：产业路径设计

→ 基于城市资源禀赋，设计3-5条产业转型路径

→ 典型路径：城市停车运营商→城市数据服务商→城市新能源服务商→城市水务一体化→城市物业服务

→ 产出：《产业转型路径与可行性分析》

💰 第四步：收入模型重构

→ 为每条产业路径搭建财务模型：收入结构、成本结构、盈利预测

→ 测算达标后的335指标改善程度

→ 产出：《经营性收入提升方案》

🏛️ 第五步：组织与能力建设

→ 设计适应市场化运营的组织架构（董事会→经营层→事业部）

→ 制定人员转型培训计划和市场化薪酬激励方案

→ 产出：《组织架构调整方案》《人员能力建设计划》

📋 第六步：达标申报与持续服务

→ 编制335达标申报材料，配合国资委核查

→ 启动后续深度服务：评级提升、数据资产入表、产业导入落地

→ 产出：《335指标达标申报书》

```

关键数据锚点

核查指标当前行业均值达标线我们的目标提升
经营性资产占比20%-25%≥30%通过经营权注入+数据资产入表，提升至35%-40%
经营性收入占比12%-18%≥30%通过停车运营+数据服务+水务，提升至35%-45%
财政补贴/利润总额60%-85%≤50%通过收入多元化，降低至40%以下
**真实案例参照**：某中部省份城投2025年启动335转型，通过将全市停车场经营权注入（评估作价15亿）、公交数据资产入表（3.2亿）、组建城市物业合资公司，8个月内经营性资产占比从21%提升至38%，经营性收入占比从13%提升至34%，财政补贴依赖度从78%降至42%。2026年3月顺利通过国资委335核查。
