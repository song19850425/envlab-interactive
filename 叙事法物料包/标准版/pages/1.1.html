1.1板块一 · 政策性资金申报

历史遗留废弃矿山生态修复示范工程

★★★★★ 100-150万
场景 SCENE

场景

中国有超过**30万公顷**的历史遗留废弃矿山，它们像大地的伤疤一样遍布各省。这些矿山大多形成于上世纪八九十年代，开采企业早已注销，"谁破坏谁治理"的原则在这里成了一纸空文。

谁在焦虑？

- 县级政府：矿坑塌陷、重金属污染、群众信访不断，但县财政一年可用财力不过几个亿，拿什么填这个坑？

- 自然资源局局长：上级督察组年年点名，整改期限一次次逼近，但治理一平方公里矿山至少需要5000万——局里全年项目经费都没这么多。

- 县委书记：矿山所在的乡镇，地没法种、人留不住、招商没人来。不治理，这块地就永远是"死资产"。

痛点 PAIN

痛点

这不是缺技术，是缺钱。更准确地说，是缺"能找到的钱"。
利益相关方核心痛点深层原因
地方政府治理资金无着落历史欠账多，谁污染谁治理原则失灵
自然资源局有项目无资金知道可以申报但不清楚申报节奏和竞争格局
省级主管部门名额有限每省限报2个，竞争激烈，选错项目等于浪费名额
咨询公司机会—**信息不对称 + 专业壁垒 = 咨询价值**

致命细节：很多县花了半年做方案，结果连省级初筛都没过。不是项目不够惨，是不知道评审专家真正看什么。

方案 SOLUTION

方案

我们不是在"写申报书"，而是在"构建一个不可拒绝的故事"。

我们的打法（六步叙事申报法）

```

① 矿山画像 → 航测+历史档案，精准锁定治理边界

② 问题诊断 → 量化生态损失：多少亩耕地损毁？多少户饮水受影响？

③ 技术方案 → "自然恢复为主、人工干预为辅"，降低造价、提高可批性

④ 效益包装 → 不只算生态账，算三本账：生态+经济+民生

⑤ 竞争定位 → 研判本省竞争格局，选择差异化赛道

⑥ 答辩演练 → 模拟专家提问，15分钟讲清楚为什么"非你不可"

```

关键数据锚点

- 门槛：治理面积 ≥ 10km²，总投资 ≥ 5亿元

- 奖补：中央财政 ≤ 3亿元/项目

- 窗口：每年3月（提前6个月启动）

- 2026年变化：每省限额从3个缩减为2个 ⚠️

价值 VALUE

价值

对客户的三重价值

价值维度具体收益量化预期
直接资金中央奖补到位2-3亿元无偿资金
土地释放治理后可利用土地10+km² 建设用地/耕地指标
政绩效应环保督察销号 + 生态产品价值实现试点省级标杆，后续政策优先倾斜

对我们的商业价值

- 单项目收入：100-150万元（前期策划+申报+答辩辅导）

- 成功率：高（成熟方法论支撑）

- 延伸机会：治理后的EOD项目策划、土地指标交易咨询

行动 ACTION

行动

现在就该做的事

时间节点行动项责任人
**现在-9月**筛选省内矿山面积≥10km²的区县，锁定3-5个目标商务团队
**10月**实地踏勘+无人机航测，完成矿山画像技术团队
**11月**出具预可研报告，向县政府做专题汇报项目负责人
