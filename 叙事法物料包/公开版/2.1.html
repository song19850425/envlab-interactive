2.1板块二 · 产业基金与融资渠道

政策性产业基金（基金债）

★★★☆☆ 80-150万
场景 SCENE

场景

截至2025年末，全国交易所累计发行基金债规模突破**6000亿元**，但真正满足'335指标'、成功拿下批文的城投平台不到发行主体的**15%**。剩下的85%，不是不想发，是过不了那道门槛。

故事从一个焦灼的周四下午开始：

某中部省份地级市城投公司，总资产72亿。董事长桌上摆着三样东西——一份发改委的产业转型批复、一份银行的拒贷通知、和一份集团2026年到期的15亿债券兑付计划表。

三天前，市领导在常务会上拍板：城投必须转型，不能再靠财政兜底。方向很明确——组建产业引导基金，用基金债撬动社会资本。但问题也很明确：交易所的门槛是资产不低于50亿、满足"335"指标——非经营性资产不超过30%、非经营性收入不超过30%、财政补贴占比不超过30%。董事长翻完自家报表，发现第一条就危险了——公益性资产占了将近四成。

更棘手的是，隔壁市同样的体量，去年花了大半年做方案，结果材料递到交易所初审就被打了回来。理由就一句话："335指标不达标，暂不符合发行条件"。八个月的心血打了水漂。

财务总监私下算了笔账：如果能发10亿基金债，年息控制在3.5%以内，期限5-7年，这笔钱作为股权出资投入产业基金——不算隐性债、不占信贷额度、不影响负债率。可如果发不成，要么走高成本的信托通道（年息8%起），要么放弃产业转型——那就等着被合并。

城投转型的窗口期就这两年。错过这波政策红利，后面只会越来越难。
痛点 PAIN

痛点

不是融不到钱，是融不到"对的"钱。
利益相关方核心痛点深层原因
城投平台资产结构不达标，335指标卡脖子历史包袱重，公益性资产占比过高
地方政府产业基金缺"种子资金"财政吃紧，无力单独出资组建基金
交易所/监管层需要合规项目但供给不足多数平台不会"包装"，方案质量参差不齐
承销券商有发行通道缺优质项目缺乏对产业逻辑的深度理解，只会做材料
咨询公司机会—**"资产结构调整 + 335达标 + 产业故事"三位一体的稀缺能力**

致命细节：很多平台以为"335指标"只是财务问题，于是找会计师事务所做报表调整。但交易所审核的是业务实质——非经营性收入不仅看数字，还要穿透看业务逻辑是否具有可持续的市场化特征。只调报表不调业务，审一轮就被识破。

咨询机会：这不是一个"代写申报材料"的生意，而是一个需要提前6-12个月介入、从资产重组到基金架构设计、再到产业投资逻辑构建的系统工程。能做到这一点的咨询团队，全国不超过20家。

方案 SOLUTION

方案

不是帮客户"凑指标"，是帮客户"长出新能力"。

五步工作流

```

① 资产体检 → 全面审计资产结构，对标335指标逐项诊断

产出：《资产结构诊断报告》+ 达标差距分析

② 资产重组 → 剥离公益性资产（政府回购/置换），注入经营性资产

关键动作：停车场收费权、砂石资源、数字资产注入

产出：《资产重组方案》+ 合规意见书

③ 基金架构设计 → "母子基金"或"直投基金"架构二选一

匹配产业方向：新能源、新材料、先进制造

产出：《产业基金设立方案》+ 投决机制设计

④ 发行材料编制 → 编制募集说明书，核心讲好三个故事：

- 资产故事（335达标证据链）

- 产业故事（投资逻辑+退出路径）

- 风控故事（还款保障+增信措施）

产出：《基金债申报全套材料》

⑤ 路演与发行 → 交易所沟通、投资者路演、利率询价

产出：发行批文 + 资金到位

```

关键技术手段

- 资产重组工具箱：公益性资产剥离的6种合规路径（政府回购、特许经营置换、机构间划转等）

- 335指标优化模型：在合规前提下，通过业务结构调整使三项指标全部达标

- 产业投资逻辑验证：借助产业智库数据，构建有说服力的投资策略和退出路径

| 📊 **关键数据锚点** |
|:--|
| **发行门槛**：总资产≥50亿，满足335指标 |
| **资金规模**：单笔3-15亿元，期限5-7年 |
| **成本区间**：年息3.0%-4.5%（视主体评级和增信）|
| **核心优势**：股权融资属性，不作为隐性债 |
| **政策窗口**：2026年交易所对产业类主体持续倾斜 ⚠️ |
| **从启动到落地**：8-14个月（资产重组周期决定） |
价值 VALUE

价值

对客户的三重价值

价值维度具体收益量化预期
融资突破低成本股权资金到位3-15亿元，年息3.0%-4.5%
资产优化335指标达标后主体评级向上2A→2A+，融资成本降低50-100bp
转型引擎产业基金撬动社会资本1:3杠杆，基金总盘可达30-50亿
**真实参照**：某中部省份地级市城投，通过资产重组将公益性资产占比从42%压至28%，成功发行8亿基金债。这笔钱作为LP出资参股了三支产业子基金，累计撬动社会资本22亿元，带动当地落地了5个制造业项目、新增就业岗位1200个。

对我们的商业价值

- 单项目收入：80-150万元（前期策划+资产重组方案+发行辅导）

- 周期：8-14个月（长期粘性高）

- 延伸机会：资产整合后的评级提升咨询、产业基金投后管理、后续增发辅导

- 壁垒：高。同时懂财务重组、产业逻辑和发行规则的团队稀缺

行动 ACTION

行动

客户开发时间线

时间节点行动项责任人
**现在**扫描省内总资产50-80亿的城投平台，筛选有产业转型诉求的目标商务团队
**第1个月**接触目标客户，进行"335指标快速诊断"（免费，2天完成）项目负责人
**第2-3个月**签署《资产重组咨询协议》，启动资产剥离与置换技术+法务团队
**第4-6个月**完成资产重组，同步设计基金架构技术团队
**第7-10个月**编制发行材料，启动交易所沟通全员
**第11-14个月**取得批文，完成发行专家团队

客户画像

- 资产规模：总资产50-100亿（低于50亿不够门槛，高于100亿通常已自建团队）

- 关键诉求：有明确的产业转型方向（新能源/新材料/先进制造），缺启动资金

- 决策人：城投董事长或分管投融资的副总

- 决策逻辑：不是买一份方案，是买"一次过审"的概率

🔑 **销售话术锚点**："您的城投总资产72亿？产业转型方向定了但钱从哪来？基金债的年息不到4%，不算隐性债、不占信贷额度——关键是帮您把资产结构调整到达标，一次上会就过。我们去年帮某地级市平台从诊断到发行，14个月拿8亿。"
