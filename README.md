# EnvLab | 逻辑驱动的环境监测数字孪生实训平台

> **Designed by Byte-Song**
> Email: jinghao.song@gmail.com

EnvLab 是一个逻辑驱动（Logic-Driven）的环境监测数字孪生实训平台，由 Byte-Song 设计。专注把真实的国标监测方法，做成可交互、做错即提醒的 PhET 级实训页。

## 🚀 已上线六大模块（MVP）
1. **总氮测定** (HJ 636-2012 碱性过硫酸钾消解紫外分光光度法)
2. **FTIR 红外光谱分析**
3. **GC-MS 气相色谱-质谱联用**
4. **HPLC 液相色谱**
5. **环境噪声监测** (声级计操作 · 等效 A 声级)
6. **PM2.5 重量法测定** (HJ 618-2011)
