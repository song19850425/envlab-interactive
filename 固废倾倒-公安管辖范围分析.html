固废倾倒 / 非法处置
公安局到底管不管？

危险废物 · 一般工业固废 · 建筑垃圾 · 逐类分析

🎯 一句话结论

管，但不是全管——而且"管不管"的判断标准，恰好需要你的检测报告来决定。

☣️ 第一类：危险废物 → 公安局 100%管辖

维度内容
法律依据刑法第338条 污染环境罪
立案门槛非法排放、倾倒、处置危废 ≥3吨 → 严重污染环境，刑事立案
管辖部门食药环侦大队（公安）→ 检察院 → 法院
与环保局关系环保局发现线索 → 移送公安 → 公安刑事立案
典型场景废酸废碱随意倾倒 / 废油漆桶非法处置 / 废矿物油卖给无资质商贩
检测需求🔥 危废鉴别 → 检测公司出报告，证明"这东西是危废"

⚠️ 2023年两高司法解释：危废数量不足3吨但涉及饮用水源地、自然保护区等敏感区域的，也可刑事立案。

🏭 第二类：一般工业固废 → 公安局 视后果而定

维度内容
法律依据固废法第102条 → 环保局行政处罚（罚款10-100万）
极端情况：刑法第338条 → 公安刑事管辖
一般情况环保局管（责令改正+罚款）→ 不涉及公安
刑事转化条件一般固废造成重大环境污染（如大量倾倒入河道致大量鱼类死亡、污染饮用水）→ 可以按污染环境罪追究刑事责任
检测需求🔥 关键！环保局/公安局需要先检测：这堆东西是"一般固废"还是"危废"？ 检出危废 → 公安管；检出一般固废 → 环保局管

💡 这里就是你最大的价值： 一堆不明废物倒在路边，环保局来了也不知道该自己罚还是移交公安——只有检测报告能给出答案。

🏗️ 第三类：建筑垃圾/渣土 → 公安局 看其中是否含危废

维度内容
纯建筑垃圾城管局管（市容环卫条例）→ 行政处罚
非法倾倒大量 → 自然资源局管（非法占地）
含危废的建筑垃圾→ 公安局管！ 装修垃圾中混入废油漆桶、废涂料、石棉瓦等危废 → 可按污染环境罪追责
非法占用农地渣土倒在农田 ≥ 10亩 → 非法占用农用地罪（公安管辖）
检测需求🔥 最核心的检测场景：装修/拆迁垃圾中是否含危废？需要采样检测才能定性。

🔄 第四类：跨省/跨市非法转运 → 公安局 联合管辖

维度内容
法律依据固废法第23条 + 刑法第338条
管辖特点跨区域非法转运固废/危废 → 公安+环保联合办案，公安负责追查运输链和犯罪链条
典型特征外省渣土/固废运入本地非法倾倒 → 涉及多个罪名（污染环境+非法经营+行贿）
公安优先度🔥 跨省转运是公安部重点关注方向，昆仑2025重点打击

🎤 给局长的汇报话术（修正版v3）

🗣️ 开门（30秒）
"汤局长，我今天就讲一件事——非法倾倒的固废到底归谁管？

现在的问题是：倾倒了一堆不明物，派出所去了、环保局去了、城管也去了，谁也说不清楚该谁管。为什么？因为没有检测报告，谁也不知道这东西到底是危废还是一般垃圾。

我们的价值就在这里——检测报告一出来，案情立刻清晰：是危废 → 食药环侦立案；不是 → 环保局行政处罚。 这事我们干了10年了。"
🗣️ 展开（2分钟）
"四个场景，四个例子：

场景一： 一堆废桶倒在河滩，派出所到了只能封控现场等上级指示。我们采样检测——桶壁残留物含苯系物 → 危废 → 刑事立案，嫌疑人当天被刑拘。

场景二： 建筑工地运出一车装修垃圾倒在农田，村民举报。我们检测——废油漆桶+废胶水瓶 → 危废混入，涉案人面临污染环境罪。

场景三： 化工厂废渣倾倒，外观看着都一样。我们检测——检出重金属超标 → 一般固废？还是危废？结论不同，执法路径完全不同。

场景四： 外省渣土车半夜开进来倾倒在废弃矿坑。我们检测——不是危废但量巨大 → 自然资源局查处非法占地；是危废 → 公安追查黑色产业链。

这四个场景有一个共同点： 案子能不能立、谁来立、怎么判，都等我们那份检测报告。"
🗣️ 收口（30秒）
"我今天不是来卖检测服务的——我是来建议打通一条'固废快检通道'：以后公安办案遇到不明废物，48小时出结论，不用等省级鉴定机构的漫长排队。

汤局长，能不能让我跟食药环侦大队长对接一次？我们先拿一个案子测一次——免费的。"

💡 对之前策略的修正

我最初说"公安局长不关心环保合规"，然后修正为"渣土车从查到判中间一步我来补"。现在可以再进一步：

  • 第一版："我给食药环侦做检测鉴定支撑"
  • 第二版："渣土车从查到判，中间那一步我来补"
  • 第三版（现在）："不明固废倾倒现场，谁来管不是局长说了算——是我的检测报告说了算。"

这个版本最狠的地方在于：它直接回答了局长在固废管理中最头疼的问题——"这到底归谁管？"。你能给他一个确切的答案。
