1️⃣ 交警大队 — 路面上的渣土车

  • 查什么：超载 / 遗撒滴漏 / 遮挡号牌 / 闯禁行(禁区通行证) / 改装加高栏板 / 重污染天气违规上路
  • 处罚依据：道路交通安全法、大气污染防治法（重污染天气违规上路）
  • 执法手段：路查路检、联合执法（城管+环保+交警）、电子警察抓拍
  • 痛点：查违法容易，但"这车拉的到底是不是建筑垃圾？含不含危废？" 交警区分不了，需要检测鉴定

2️⃣ 食药环侦大队 — 渣土去哪儿了

  • 查什么：非法倾倒建筑垃圾/渣土 → 占用农用地 / 污染环境 / 非法设置消纳场
  • 法律红线：
    • 非法倾倒普通建筑垃圾 ≥ 10亩 → 非法占用农用地罪
    • 倾倒含有危废的装修垃圾（油漆桶/废胶/石棉）→ 污染环境罪（刑法338条）
    • 跨区域转运建筑垃圾 → 可能涉及非法处置固体废物
  • 核心需求：👈 这就是检测公司进场的点了！ 食药环侦需要知道"这东西到底是什么废物"才能定性

3️⃣ 辖区派出所 — 末端处置

  • 渣土车夜间偷倒、村民举报、施工噪音 → 派出所先出警 → 发现涉及环境犯罪 → 移交给食药环侦
  • 派出所日常巡查发现废品收购站/废品站违规收储建筑装修垃圾

🎯 这就厉害了——你的54项业务，正好卡在3条线的交叉点

💡 核心洞察
渣土车从"上路"到"倾倒"到"定性"到"定罪"——
前面的执法环节（查超载/查遗撒/查倾倒）交警和辖区派出所都能做，但到了"这东西到底是什么废物、有没有毒、值不值得立案"这一步，他们都卡住了。

这一步，就是你的检测公司。

具体能帮什么：

  • 危废鉴别 装修/拆迁垃圾中是否含危废（废油漆桶、废涂料、石棉瓦、废胶）→ 决定能否以污染环境罪立案
  • 固废属性鉴别 倾倒物是建筑垃圾、一般固废还是危废？→ 确定执法依据（治安处罚还是刑事立案）
  • 土壤检测 非法倾倒点位的土壤是否已受污染、污染范围多大 → 损害评估和修复费用计算
  • 总量核算 倾倒方量测算、成分分析 → 量刑依据

交警在路上查渣土车，食药环侦在查非法倾倒，但走到关键一步都卡住了——这东西到底是什么废物？有没有毒？够不够立案标准？

我们就是干这个的——做固废鉴别和危废鉴定，帮公安局把案子走通最后那一公里。"
🗣️ 展开话术（2分钟）
"这个事有多现实？给您举个例子：

一堆渣土倒在农田里，派出所出警，但只能按治安案件处理——如果里面混了废油漆桶、废胶水瓶、石棉瓦，那就不是治安案件了，是污染环境罪，刑期和罚金完全不是一个量级。

唯一的区别在于——有没有检测鉴定报告证明它含危废。 这份报告我们能出。

再一个，交警在查渣土车遗撒的时候，如果当场发现撒漏物疑似危废，也能第一时间取样送检，不用等到出事故了才事后追责。"

能不能让食药环侦大队长跟我对接一次，我们先用一个真实案子测一测？免费。"

💡 修正后的整体判断

我之前说"公安局长不关心环保"是错的。实际上：

  • 渣土车管理是交警的核心业务 — 超载、遗撒、禁行、扬尘，天天在抓
  • 非法倾倒建筑垃圾是食药环侦的重点 — 占用农用地、污染环境
  • 你卡在关键环节 — 交警只查"违法"，食药环侦只查"犯罪"，中间的"这东西是什么废物"没人干——你来干

一句话修正后的策略：
从"我给食药环侦做检测鉴定支撑" → "渣土车从查到判，中间那一步我来补"
前者是专业功能，后者是办案链条——局长更好理解。
